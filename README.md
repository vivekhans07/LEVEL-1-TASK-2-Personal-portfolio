# 💼 Vivek Hans Portfolio Website

A modern, responsive personal portfolio website built using **HTML5** and **CSS3**. This portfolio showcases my background as an Electronics & Communication Engineering student, web developer, embedded systems enthusiast, and founder of LearnWell Tutorials.

## 🌟 Features

- Responsive design for desktop, tablet, and mobile
- Sticky navigation bar
- Professional hero section
- About Me section with certifications
- Technical skills categorized by domain
- Projects showcase
- Education timeline
- Contact section
- Modern UI with smooth scrolling
- Pure HTML & CSS (No frameworks)
